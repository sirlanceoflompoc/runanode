<h3 align="center">
  A node wallet that can stake CENNZ tokens
</h3>
<p align="center">
  rUNanode is desktop version of node in Cennznet network
</p>

## Download desktop software

[runanode.io][runanode.io]

## Get started

```bash
brew install jq  // Or similar appoach on Linux and Windows
npm i
npm run dev
```
